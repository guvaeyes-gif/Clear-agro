# Checklist LGPD

- [ ] Finalidade do tratamento documentada.
- [ ] Base legal definida.
- [ ] Dados pessoais estritamente necessarios.
- [ ] Campos sensiveis mascarados/anomimizados quando aplicavel.
- [ ] Compartilhamento com terceiros validado.
- [ ] Retencao e descarte definidos.
- [ ] Responsavel e data de aprovacao registrados.
