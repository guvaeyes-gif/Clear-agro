# 12_integracoes_agent - Fontes de Dados

## Fontes principais
- APIs externas
- Webhooks
- Logs de integracao
- Credenciais seguras

## Requisitos de uso
- Toda consulta deve ter timestamp e referencia da fonte.
- Divergencia entre fontes deve ser sinalizada explicitamente.
- Nao usar fonte nao homologada sem aprovacao de system_admin.
