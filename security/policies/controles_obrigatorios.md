# Regras de Governanca Obrigatorias

1. Toda pasta deve ter dono responsavel definido.
2. Todo dashboard deve ter fonte, dono e finalidade.
3. Todo agente deve ter escopo, fontes e regras de acesso.
4. Toda automacao critica deve registrar log e erro.
5. Toda integracao deve ter documentacao e politica minima de seguranca.
6. Mudancas estruturais devem ser registradas em roadmap, migracoes e logs.
